# ScamShield AI

ScamShield AI is an AI-powered cybersecurity platform that helps users detect phishing attempts, scam messages, suspicious links, impersonation, and emotional manipulation using Gemini AI.

## Features

- Scam probability analysis
- Threat level detection
- Phishing URL analysis
- Psychological manipulation detection
- AI-generated explanations
- Cyber safety recommendations
- Modern cybersecurity-inspired UI

## Tech Stack

- Lovable
- React
- TailwindCSS
- Gemini API
- Framer Motion
- TypeScript
- Vite

## How It Works

Users paste suspicious messages or phishing links into the analyzer. Gemini AI then evaluates the content and returns:
- scam probability score
- manipulation tactics
- scam category
- safety recommendations
- human-friendly explanations

## Inspiration

Online scams are increasing rapidly across SMS, emails, and social media platforms. ScamShield AI was built to improve cybersecurity awareness and help users identify manipulation tactics before becoming victims.
